# 🧠 AI Text Analyzer App

## 🚀 About Project
This is a Streamlit-based AI web app that performs sentiment analysis, keyword extraction, and text visualization.

## ✨ Features
- Sentiment Analysis (VADER NLP)
- Keyword Extraction
- Pie & Bar Charts
- File Upload Support
- Simple UI Dashboard

## 🛠️ Tech Stack
- Python
- Streamlit
- NLTK
- Matplotlib
